Hot Category News displays a selected number of posts from a specific category.

== Description ==

Hot Category News plugin by [HotThemes](http://hot-themes.com/ "HotThemes") displays a selected number of posts from a specific category in form of a widget. You can select number of posts to be fetched from a category. From the widget parameters, you can enable or disable post titles, post dates,  and limit the number of words. Also, it's possible set posts ordering or randomize the posts.

== Installation ==

1. Install plugin as usually from WordPress.org. You can also manually download it, unpack and upload `plugin_hot_category_news` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Publish Hot Category News widget into the available widget sidebars.
4. To get some posts displayed, you should pick a category by entering the category slug or ID.
5. In the widget options, you can see other parameters that defines the widget's output. Experiment with these options to configure the widget for your site.

== Frequently Asked Questions ==

= What to do with plugin after installation? =

Go to Appearance > Widgets and drop Hot Category News widget into any widget position. Then open it to see the widget options.

= My widget doesn't show any data? =

Make sure you entered a valid category slug or ID in widget parameters.
